1、常用编辑器有WebStorm、Sublime、VS Code等，推荐WebStorm，因为功能比较强大，配置少。

3、WebStorm快捷键

* tab键的使用；
* Ctrl + D  复制本行；
* Ctrl + X  删除本行
* Ctrl + F  单文件内搜索；
* Ctrl + Shift + F  全局搜索；
* Ctrl + Alt + L  格式化代码；
* Ctrl + Shift +/  多行注释；
* .......
